Breast Cancer Logistic Regression

By Stephen Situ

This application uses a logistic regression model to predict the likelihood of a patient having breast cancer. The model was developed using data from the Breast Cancer Prediction dataset, which is available on Kaggle. You can find more information about the dataset here. The model development can be found here. The model takes in various inputs such as area, symmetry, concavity, and other measurements to make its prediction. We hope that this application will be useful for medical professionals and patients alike.
